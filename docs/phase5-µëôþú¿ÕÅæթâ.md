## 五、性能基线

| 指标 | 目标 | 备注 |
|------|------|------|
| 安装包大小 | <10MB | Tauri 典型体积 5-8MB |
| 空闲 CPU | <3% | 精灵图无动画 + 漫游暂停 |
| 活跃 CPU | <8% | 全动画 + 移动 + 监控 |
| 空闲内存 | <50MB | WebView + Rust backend |
| HTTP 延迟 | <50ms | Hook 接收 |
| MCP 延迟 | <100ms | JSON-RPC 调用 |
