> **工期**: **10 个工作日** | **里程碑**: MCP 服务完整、可被外部调用、自定义事件框架可用

实现 MCP 深度集成：**MCP memory + resources 持久化**、标准 MCP tools 完善、**自定义事件框架**（event bus）供第三方集成、**皮肤/主题/插件系统**基础。最终版本标记 `v0.1.0`。

## 六、风险项

| 风险 | 等级 | 缓解措施 | 责任人 | 回退方案 |
|------|------|----------|--------|----------|
| MCP memory 格式向前兼容性 | 中 | 使用 semver 版本字段 + 自动迁移 | 开发 | 清空 memory 重新构建 |
| EventBus 并发竞争 | 中 | 使用 `tokio::sync::RwLock` 保护共享状态 | 开发 | 降级为单线程处理 |
| 皮肤切换时动画引擎状态不一致 | 中 | 切换前暂停动画 → 加载资源 → 恢复动画（原子操作） | 开发 | 重启动画引擎 |
| 插件系统安全风险（任意代码执行） | **高** | 插件使用 WASM 沙箱或限制 API 权限；设置权限白名单 | 开发 | 不支持第三方插件（仅皮肤） |
| 第三方集成文档不完善导致集成困难 | 低 | Day 7 专门产出 4 份文档 + 代码示例 | 开发/设计 | 增加 FAQ 文档 |
