# 桌宠终端命令速查

> 所有命令在 **PowerShell**（项目根目录 `d:\project file\Desktop_Pet`）下直接复制粘贴执行。

## 一、项目启动 / 构建 / 测试命令

### 0. 前置条件（首次执行）
```powershell
# 进入项目目录
cd "d:\project file\Desktop_Pet"

# 安装 Rust（若未安装）
winget install --id Rustlang.Rustup -e
# 安装 Node.js（用于前端依赖，按需）
winget install --id OpenJS.NodeJS.LTS -e
```

### 1. 开发环境（开发热跑）
```powershell
# Rust 增量编译 + 自动运行（推荐）
cd "d:\project file\Desktop_Pet\src-tauri"
cargo run

# 仅检查代码（不生成可执行文件，速度快）
cd "d:\project file\Desktop_Pet\src-tauri"
cargo check

# 实时编译日志
cd "d:\project file\Desktop_Pet\src-tauri"
cargo build 2>&1 | Tee-Object -FilePath ..\build.log
```

### 2. 测试环境（单元测试 / 集成测试）
```powershell
# 跑全部测试
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test

# 跑指定测试（如运动系统）
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test movement

# 跑指定测试函数
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test test_animation_name_all_states

# 显示 println! 输出
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test -- --nocapture

# 跑指定模块
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test state::
```

### 3. 生产环境（正式打包）
```powershell
# Release 构建（优化后二进制）
cd "d:\project file\Desktop_Pet\src-tauri"
cargo build --release

# Release 并运行单元测试
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test --release

# 清理旧的构建产物
cd "d:\project file\Desktop_Pet\src-tauri"
cargo clean

# 一键打包安装包（需先配置 tauri.conf.json 中的 bundle）
cd "d:\project file\Desktop_Pet\src-tauri"
cargo tauri build
```

### 4. 实用工具
```powershell
# 更新依赖
cd "d:\project file\Desktop_Pet\src-tauri"
cargo update

# 依赖树
cd "d:\project file\Desktop_Pet\src-tauri"
cargo tree

# 代码格式化
cd "d:\project file\Desktop_Pet\src-tauri"
cargo fmt

# Clippy 静态检查
cd "d:\project file\Desktop_Pet\src-tauri"
cargo clippy --all-targets -- -D warnings

# 启动桌面快捷脚本
cd "d:\project file\Desktop_Pet"
.\start-desktop-pet.bat
```

## 二、行为动作触发命令

> 桌宠启动后默认监听 `http://127.0.0.1:9527`，以下命令可直接测试。
> MCP 路径：`/mcp`，HTTP 路径：`/api/...`

### 1. MCP JSON-RPC 协议（推荐，CLI 工具集成方式）

**显示自定义气泡（pet_show）**
```powershell
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_show\",\"arguments\":{\"msg\":\"Hello from terminal!\"}}}'
```

**强制播放动画（pet_play）**
```powershell
# 挥手
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_play\",\"arguments\":{\"state\":\"waving\"}}}'

# 跳跃
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_play\",\"arguments\":{\"state\":\"jumping\"}}}'

# 思考
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_play\",\"arguments\":{\"state\":\"thinking\"}}}'
```

**询问用户（pet_ask / pet_get_user_input）**
```powershell
# 自由文本提问
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_ask\",\"arguments\":{\"question\":\"今天写点啥?\"}}}'

# 选项选择
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_ask\",\"arguments\":{\"question\":\"选一个动作\",\"options\":[\"wave\",\"jump\",\"dance\"]}}}'

# 阻塞等待（confirm 类型）
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/call\",\"params\":{\"name\":\"pet_get_user_input\",\"arguments\":{\"prompt\":\"确定要继续吗?\",\"type\":\"confirm\"}}}'
```

**列出所有可用工具**
```powershell
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"tools/list\"}'
```

### 2. HTTP REST API（部分状态直接触发）

**触发状态变化（模拟适配器 hook 事件）**
```powershell
# 工作中
curl -X POST http://127.0.0.1:9527/api/state `
  -H "Content-Type: application/json" `
  -d '{\"state\":\"running\",\"tool\":\"Read\"}'

# 完成
curl -X POST http://127.0.0.1:9527/api/state `
  -H "Content-Type: application/json" `
  -d '{\"state\":\"review\"}'

# 失败
curl -X POST http://127.0.0.1:9527/api/state `
  -H "Content-Type: application/json" `
  -d '{\"state\":\"failed\"}'

# 庆祝
curl -X POST http://127.0.0.1:9527/api/state `
  -H "Content-Type: application/json" `
  -d '{\"state\":\"celebrating\"}'
```

**适配器状态查询**
```powershell
# 查询当前适配器
curl http://127.0.0.1:9527/api/adapter/status

# 立即重新扫描
curl -X POST http://127.0.0.1:9527/api/adapter/scan

# 切回自动模式
curl -X POST http://127.0.0.1:9527/api/adapter/auto
```

**运动模式控制**
```powershell
# 查询当前运动模式
curl http://127.0.0.1:9527/api/movement/mode

# 切换模式
curl -X POST http://127.0.0.1:9527/api/movement/toggle
```

### 3. 用户消息 / 输入响应

**向桌宠发送文本消息**
```powershell
curl -X POST http://127.0.0.1:9527/api/user/message `
  -H "Content-Type: application/json" `
  -d '{\"value\":\"开始新的任务\"}'
```

**响应桌宠的提问（替代气泡交互）**
```powershell
# 文本回答
curl -X POST http://127.0.0.1:9527/api/user/input `
  -H "Content-Type: application/json" `
  -d '{\"value\":\"我的回答\",\"type\":\"text\"}'

# 确认 yes/no
curl -X POST http://127.0.0.1:9527/api/user/input `
  -H "Content-Type: application/json" `
  -d '{\"value\":\"yes\",\"type\":\"confirm\"}'

# 取消
curl -X POST http://127.0.0.1:9527/api/user/input `
  -H "Content-Type: application/json" `
  -d '{\"value\":\"__cancel__\",\"type\":\"text\"}'
```

## 四、常见问题排查

```powershell
# 1. 端口被占用（9527 是默认 HTTP/MCP 端口）
netstat -ano | findstr :9527
# 杀掉占用进程（替换 <PID>）
Stop-Process -Id <PID> -Force

# 2. 重新编译（依赖变更后）
cd "d:\project file\Desktop_Pet\src-tauri"
cargo clean
cargo build

# 3. 查看运行日志
cd "d:\project file\Desktop_Pet\src-tauri"
cargo run 2>&1 | Tee-Object -FilePath ..\run.log

# 4. MCP 协议握手测试
curl -X POST http://127.0.0.1:9527/mcp `
  -H "Content-Type: application/json" `
  -d '{\"jsonrpc\":\"2.0\",\"id\":1,\"method\":\"initialize\"}'

# 5. 列出所有 task 子任务
cd "d:\project file\Desktop_Pet\src-tauri"
cargo test -- --list
```
